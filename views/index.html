MicroService TimeStamp

Directions

  1. Pass a string as a parameter in the URL, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
  2. If it does, it returns both the Unix timestamp and the natural language form of that date as JSON.
  3. If it does not contain a date or Unix timestamp, it returns null for those properties as JSON.

Example Usage

https://timestamper-api-fcc.herokuapp.com/October27,1910

https://timestamper-api-fcc.herokuapp.com/-1867622400"

Example Output

{unix: "-1867622400", natural: "October 27, 1910"}
